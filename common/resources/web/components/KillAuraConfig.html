Kill Aura

Automatically attacks entities in range with current item in hand.
Delay Method
Tick Interval:
How often kill aura attacks. 1 = fastest (attack every tick), 20 = attack once a second, and so on. In other words: attack every X ticks.
Extra Ticks:
Wait additional ticks once cooldown passed. Can be negative; in this case, the module will attack before cooldown ends.
Max Range
Limit target coverage by angle
Delta horizontal angle:
Delta vertical angle:
In degrees. Value 180 means no limit. But it is better to leave fields empty for this purpose.
For every possible target, the module calculates how far you need to rotate to attack the target. If this angle exceeds the specified limit, the module will not attack this target.
Note: the module is not smart enough to attack on edges of a hitbox. It always attacks the center of the entity's bounding box. Angle is measured relative to this point.
The module attacks all targets in range. Only the first target (closest) receives full damage.
